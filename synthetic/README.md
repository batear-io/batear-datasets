# Synthetic Data

Pure, algorithmically generated signals for DSP calibration and unit testing.

## Subdirectories

| Directory | Description |
|---|---|
| `sine-sweeps/` | 10 Hz to 100 kHz linear and logarithmic frequency sweeps |

## Usage

These samples provide known-frequency references for validating filter
responses, FFT bin accuracy, and anti-aliasing behavior at the 16 kHz
production sample rate.
